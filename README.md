SoraAI

A full stack AI powered web application built using modern technologies. It integrates a frontend interface with a backend server and external services to deliver an intelligent and scalable user experience.

📌 Overview

This project was originally developed on Replit and later migrated to GitHub for version control and deployment.

SoraAI includes:

A modern frontend built with Vite and React
A backend server using Node.js and Express
Integration with external APIs and services
Modular and scalable architecture


Project Structure
client/     → Frontend (React + Vite)
server/     → Backend (Node + Express)
shared/     → Shared logic between frontend and backend
